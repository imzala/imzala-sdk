# İmzala SDK

İmzala dijital imza platformu için resmi SDK seti + gömülü imza (embedded signing).

> ⚠️ Geliştirme aşamasında. Yayın öncesi avukat onayı + kapsam onayı tamamlanacaktır.

## Paketler
- `@imzala/embed` — tarayıcı gömülü imza widget'ı (iframe)
- `@imzala/embed-react` — React sarmalayıcı
- (yakında) `@imzala/node`, PyPI `imzala`, NuGet `Imzala`, Packagist `imzala/imzala-php`, Maven `org.imzala:imzala-java`
